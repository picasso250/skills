everything-file-search

Fast local file search with Everything.

Use this skill to query the Everything HTTP service and return full file paths for local files quickly, including Chinese file names.

Everythingfile searchlocal paths

What It Does

Searches the local system with Everything and returns full paths instead of partial matches.

Best For

Finding scripts, assets, downloads, generated files, and exact file locations fast.

Install

  1. git clone --depth 1 https://github.com/picasso250/skills.git ~/tmp/picasso-skills
  2. cp -r ~/tmp/picasso-skills/everything-file-search ~/.agents/skills/
  3. Check the installed skill directory for extra README instructions.
