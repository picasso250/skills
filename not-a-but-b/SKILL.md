# Skill: 去「不是A，而是B」句式

## 问题定义

这是一个懒惰的修辞拐杖。写作者（尤其是 AI）用「否定一个靶子 + 亮出正确答案」的结构来假装有深度，实际上是回避了对事物本身的直接描述。

重复使用会让文本产生严重的 AI 腔：**听起来像在论证，实际上在绕路。**

## 常见变体清单

以下是同一底层结构的不同包装。每一种都要查。

| 句式 | 例 |
|------|----|
| 不是A，是B | 不是软弱，是慈悲 |
| 不是A。是B（拆句） | 他靠的不是武功。是他的坚定 |
| 不是要A，是要B | 不是要降妖，是要渡人 |
| 不是因为A，是因为B | 不是因为打不过，是因为不想打 |
| 不是靠A，是靠B | 不是靠武力，是靠定力 |
| 不在于A，在于B | 不在于快，在于准 |
| 并非A，而是B | 并非天真，而是笃定 |
| 表面上A，实际上B | 表面上懒散，实际上清醒 |
| A，其实B | 听着硬。其实是感情压成的沉默 |
| 没有A，是B | 没有犹豫。是笃定 |
| 冲的不是A，是B | 冲的不是他的武力。是他的坚定 |
| 不A，B | 不讲面子，讲效率 |
| 不在于A，在于B | 不在于她多能打，在于她多想赢 |

## 核心判断标准

只要一句话的结构是：**先否定一个选项 + 再亮出真正的答案**，就中招了。

无论否定词是「不是」「并非」「没有」「不在于」「不是靠」「不是因为」「不(动词)」，无论拆成一句还是两句，无论连接词是「而是」「而是说」「其实是」「实际上是」——**一律视为同一种毛病。**

## 写作标杆

不要方法论。记住三个人的写法就够了。

### 鲁迅

冷，硬，不走弯路。一句话就是一个钉子，不给自己设靶子再打。

```
有问题的写法：他不是天真，他是笃定。
鲁迅会怎么写：他信。旁人说破天，他信他的。
```

鲁迅写人从来不「不是A，是B」。他写范爱农：「他高大，眉目很威严，说话的声音是钝的，不大好听。」——直接给事实，让读者自己感受。

### 朱自清

从具体事物起笔。写一件事就是一件事，不拿它去对比另一件事。

```
有问题的写法：父亲不是不关心我，他只是不善表达。
朱自清会怎么写：《背影》里父亲爬月台买橘子。全程没有一句「他爱我」，但每一个动作都是。
```

朱自清的办法：写那个动作，写那个场景，让细节自己说话。

### 汪曾祺

最干净。只写「是什么」和「做什么」，不写「不是什么」。

```
有问题的写法：这个女人不是普通人，她是个有故事的人。
汪曾祺会怎么写：她每天在巷口摆摊。下雨也摆。\n写一个人，让读者看见她在做什么就够了。
```

汪曾祺写金冬心：「他有一个脾气，爱起早。每天天不亮就起来，在院子里走走。」——不解释这个人是什么性格，一个动作全透出来了。

## 检验标准

改完之后自己读一遍。

读到某句话，觉得「这话像从 AI 嘴里说出来的」——那就是没改干净。砍掉重写。

读到整段话，觉得自己像是在听一个人说话，有呼吸、有停顿、有时候话说一半就懂了——那就对了。

不要检查句式，检查感受。

## 验证方法

写完后全文 grep 以下关键词。任何一个命中都说明有残留。

- `不是` — 最核心的否定词，必须逐处判断
- `不在于`
- `并非`
- `而不是`
- `没有(动词)` — 如「没有选择A，而是B」
- `其实` — 常见于「A，其实B」变体
- `表面上`
- `实际上`
- `冲(着)的不是` / `靠的不是` / `图的不是` / `要的不是`

**特别注意拆成两句话的情况**——先一个句号否定，下一句肯定。这种在 grep 里只会命中「不是」一词，容易被漏掉。必须逐句通读。

## 使用场景

- 台词本自我审查
- 文案写作后查 AI 腔
- 给他人文稿做润色
- 任何需要「听起来像人写的」的文字

## 原则

1. **先删再补。** 把所有「不是A，而是B」句子先砍成只剩后半句，看信息还在不在。通常都在，而且更短。
2. **不要试图用替换词蒙混。** 把「不是」换成「并非」、把「而是」换成「其实是」——只是换件马甲，病根没动。
3. **每句话结构独立。** 检查相邻两句不要重复使用同一句式模板。
4. **用画面代替判断。** 如果能写一个具体动作、一个场景、一句对话，就不要写抽象概念。
