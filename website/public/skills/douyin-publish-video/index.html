douyin-publish-video / 抖音发布视频

将本地视频发布到抖音

适合把手动网页发布动作变成可重复 CLI，或连接已登录浏览器会话完成标题、简介和最终发布。

抖音发布Douyin automationvideo publishing

用途

上传本地视频，复用发布流程，减少重复网页操作。

适合场景

批量内容发布、发布前人工确认、已登录浏览器会话自动化。

安装

  1. git clone --depth 1 https://github.com/picasso250/skills.git ~/tmp/picasso-skills
  2. cp -r ~/tmp/picasso-skills/douyin-publish-video ~/.agents/skills/
  3. 安装后继续查看该目录内 README 或 SKILL.md 的补充说明。
